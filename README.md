# 🎓 Student Management System in C

## 📖 About
This is a beginner-friendly Student Management System written in C. The program accepts student details, calculates total marks and percentage, assigns grades for each subject, and displays an overall grade with remarks.

This project was created as part of my C programming learning journey.

---

## ✨ Features

- Enter student's name
- Enter roll number
- Enter marks for 5 subjects
- Calculate total marks
- Calculate percentage
- Display subject-wise grades
- Display overall grade
- Display remarks based on performance
- Validate marks (0–100)

---

## 🛠️ Technologies Used

- C Programming Language
- Coding.c app

## 📋 Grading System

| Marks | Grade |
|-------:|:-----:|
| 90–100 | A+ |
| 80–89 | A |
| 70–79 | B+ |
| 60–69 | B |
| 50–59 | C |
| 41–49 | D |
| 0–40 | F |
